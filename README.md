# Poncio Landing Reparada

Landing estatica lista para ejecutar.

## Ejecutar
Abrir `index.html` en el navegador o servir la carpeta con cualquier servidor estatico.

## Estructura
- `index.html`
- `assets/css/styles.css`
- `assets/js/main.js`
- `assets/img/`
- `assets/video/`

## Incluye
- Logo Poncio en version negativa para header y footer.
- Hero con video de fondo y overlay oscuro.
- Boton flotante de WhatsApp.
- Imagenes de servicios, equipo y secciones principales.
- Responsive desktop, tablet y mobile.
